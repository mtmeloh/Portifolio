Sobre Mim

Sou estudante de Sistemas de Informação com foco em desenvolvimento back-end e construção de aplicações web escaláveis. Tenho experiência prática com Java 17+, Spring Boot, Spring Data JPA e desenvolvimento de APIs RESTful, aplicando arquitetura em camadas, organização de código e boas práticas de versionamento com Git.

Possuo conhecimento em bancos de dados relacionais como MySQL e PostgreSQL, modelagem de dados e integração entre sistemas. Também tenho familiaridade com HTML, CSS e JavaScript, o que me permite compreender o fluxo completo de uma aplicação web.

Tenho perfil analítico, facilidade para resolver problemas e disciplina para aprendizado contínuo. Hoje busco minha primeira oportunidade como Desenvolvedor Java Júnior para contribuir tecnicamente com o time, evoluir profissionalmente e participar da construção de soluções eficientes e bem estruturadas.
